 JS-DOS Browser Emulator

Browser-based DOS game emulator built with JavaScript and Node.js.  
This is a learning project focused on understanding how emulation can run inside a modern browser.

 About the project
The goal of this project is to experiment with:
- running DOS games in the browser using JS-DOS
- interaction between frontend and backend
- serving static files and handling basic server logic

This project is not production-ready and is created for learning purposes.
